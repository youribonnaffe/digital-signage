Simple digital signage

An easy way to display web content on remotely devices!

1. Pick a network name, such as and access the admin interface

2. On your remote device, pick a client's name such as and access the client interface

3. In the admin interface, set the URL the remote device should display.

Iframes are used to display the webpage on the client side. Therefore the header X-Frame-Options must not be set to sameorigin.
Nothing is enforced in term of security, anyone can guess a network's name to access the admin page.
